# Setup Guide

## Prerequisites

- Git installed on your machine
- A text editor or IDE (VS Code recommended)
- Basic command line knowledge

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/Diarmaid26/Training.git
cd Training
```

### 2. Explore the Structure

```bash
ls -la
```

You'll see:
- `docs/` - Documentation
- `modules/` - Training modules
- `exercises/` - Practice exercises
- `projects/` - Project files
- `solutions/` - Solution references
- `tests/` - Test files

### 3. Get Started

Read through the [README.md](../README.md) and explore the modules directory.

## Environment Setup

Depending on the training focus, you may need:

- **Python**: `python --version`
- **Node.js**: `node --version`
- **Git**: `git --version`
