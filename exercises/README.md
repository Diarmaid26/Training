# Exercises

Practice problems organized by difficulty level.

## Difficulty Levels

### 🟢 Beginner
Foundational exercises for getting started.

### 🟡 Intermediate
Mid-level challenges to strengthen your skills.

### 🔴 Advanced
Complex problems for experienced learners.

## How to Use

1. Start with beginner exercises
2. Solve problems independently
3. Test your solutions
4. Review solutions for reference
5. Progress to the next level

## Tips for Success

- Read the problem carefully
- Plan your approach before coding
- Test with edge cases
- Refactor for clarity
- Learn from solutions
